
# My Spring Boot Project

## Overview
This is a Spring Boot application project.

## Getting Started

### Prerequisites
- Java 11 or higher
- Maven 3.6+

### Installation
1. Clone the repository
2. Navigate to the project directory
3. Run `mvn clean install`

### Running the Application
```bash
mvn spring-boot:run
```
